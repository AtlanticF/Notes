# 计算机体系结构
	现代计算机都是基于 Von Neumann 体系结构的，不管是嵌入式系统、PC还是服务器。这种体系结构主要由 CPU（central process unit）、Memory、Input and Output

![[computer-bus.jpeg]]

- CPU
- Memory
	- ROM (Read Only Memory) 只读存储
		- PROM (Programable Read Only Memory) 可编程只读存储器
		- EPROM (Erasable Progamable Read Only Memory) 可擦除可编程只读存储器。可通过 `强紫外线` 来擦出数据，它是一组 `浮栅晶体管`，数据可保持 10 ～ 20 年。老式的 BIOS 芯片一般是 EPROM
		- EEPROM (Electrically Erasable Programmable Read Only Memory) 电可擦除可编程只读存储器。相比 EPROM，EEPROM 不需要用紫外线照射，也不需取下，就可以用特定的电压，来抹除芯片上的信息，以便写入新的数据。FLASH 属于 EEPROM
	- RAM (Random Access Memory) 随机访问存储
- I/O


# CPU
CPU 由 `运算单元`, `数据单元`, `控制单元` 组成

`运算单元` 对操作数进行加 、减、位移等运算操作
`数据单元` 对操作数和运算结果进行保存。包括 CPU 内部的缓存和寄存器组
`控制单元` 发出相关计算指令，从数据单元获取操作数到运算单元进行运算，将结果保存到数据单元的寄存器中
